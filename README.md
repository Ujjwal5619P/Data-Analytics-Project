# Heart Disease Data Analysis using Tableau

## 📌 Project Overview

This project analyzes heart disease health indicators using interactive visualizations built in Tableau.
The objective is to identify important risk factors such as smoking, obesity (BMI), diabetes, and physical inactivity that contribute to cardiovascular diseases.

The dashboard and story help healthcare professionals, researchers, and policymakers understand patterns in heart disease and make informed preventive decisions.

---

## 🎯 Project Objectives

* Analyze heart disease patterns using visual analytics.
* Identify relationships between lifestyle habits and heart disease.
* Discover risk factors such as smoking, alcohol consumption, diabetes, and physical inactivity.
* Create interactive dashboards and story points for better decision-making.

---
